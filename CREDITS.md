# Credits

RPMWorks is built on the shoulders of a lot of excellent open source work. These are the projects that make it possible.

## Backend

| Project | What it does |
|---|---|
| [FastAPI](https://fastapi.tiangolo.com/) | Web framework for the REST API |
| [Starlette](https://www.starlette.io/) | ASGI foundation that FastAPI is built on |
| [Uvicorn](https://www.uvicorn.org/) | ASGI server |
| [SQLAlchemy](https://www.sqlalchemy.org/) | Database ORM |
| [Pydantic](https://docs.pydantic.dev/) | Data validation and settings |
| [Paramiko](https://www.paramiko.org/) | SSH and SFTP client |
| [psycopg](https://www.psycopg.org/) | PostgreSQL driver |
| [passlib](https://passlib.readthedocs.io/) | Password hashing |
| [argon2-cffi](https://argon2-cffi.readthedocs.io/) | Argon2 password hashing algorithm |
| [python-jose](https://python-jose.readthedocs.io/) | JWT tokens for authentication |
| [cryptography](https://cryptography.io/) | Cryptographic primitives |
| [PyNaCl](https://pynacl.readthedocs.io/) | NaCl cryptography bindings (used by Paramiko) |
| [python-multipart](https://andrew-d.github.io/python-multipart/) | Multipart form data parsing |
| [anyio](https://anyio.readthedocs.io/) | Async I/O abstraction |
| [click](https://click.palletsprojects.com/) | Command-line interface utilities |
